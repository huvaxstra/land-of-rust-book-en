# Land of Rust – English Edition (mdbook)

This repository contains the English version of the **Land of Rust** book, built with **mdbook**. It is derived from the original [land-of-rust](https://github.com/huvaxstra/land-of-rust) project, reorganized as a clean, navigable online book for Rust learners.

## Features
- ✅ Beginner‑friendly explanations with real‑world examples
- ✅ Fully compatible with `mdbook` – search, theming, and code highlighting
- ✅ Covers core Rust concepts: ownership, lifetimes, concurrency, async, and more
- ✅ Ready to be deployed on GitHub Pages or any static hosting
